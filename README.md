# Navratri Bazaar

A festive landing page built for a Navratri-themed e-commerce experience.

## ✨ About

This project is a responsive, modern festival storefront designed for Navratri shopping. It includes vibrant styling, mobile-friendly navigation, polished typography, and interactive UI elements for search, cart, and contact actions.

## 🚀 Features

- Responsive hero section with festival-themed visuals
- Curved navigation buttons and mobile menu support
- Search input with interactive toast feedback
- Add-to-cart interactions with live cart count
- Special offers and product showcase sections
- Clean contact form and footer with subscription prompt
- Consistent festival color palette and typography

## 🛠️ Technologies Used

- HTML5
- CSS / Tailwind CSS CDN
- JavaScript
- Google Fonts (`Inter`, `Playfair Display`)

## 📌 How to Use

1. Open `Web Development/index 6.html`
2. View in browser
3. Interact with menu, search, and add-to-cart buttons
4. Customize images, colors, or product text as needed

## 💡 Why This Page

This landing page is ideal for festival marketing with:
- Strong visual hierarchy
- Smooth user interactions
- Clear product and offer presentation
- Mobile-friendly layout
